Claude AI

Paste a recipe from any website, a photo caption, a text message — any format. Claude will extract and structure it for your cookbook automatically.

How to use with Claude for Chrome

When browsing a recipe site, open Claude for Chrome and say:
"Summarize this recipe into: Name, Category, Cook time, Servings, Difficulty, Ingredients (one per line), Instructions, Notes"

Then paste Claude's response into the box above and click Extract & save.

Don't have a Railway URL yet?
1. Go to railway.app and create a free account
2. New Project → Deploy from GitHub → select the cookbook-api repo
3. Add a PostgreSQL database, then copy the public URL here
